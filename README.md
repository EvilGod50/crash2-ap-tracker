## Crash Bandicoot 2 Poptracker
Poptracker for Crash Bandicoot 2 AP integration

Developed by @Evil God. If you run into any bugs please report them on the Crash Bandicoot 2 thread/channel on the AP Discord Server or on this github's issues page

Contains full item and location tracking aswell as autotracking for all of Crash Bandicoot 2 AP features

Credit to Stripes007's [Poptracker Pack Builder](https://github.com/StripesOO7/poptracker-pack-builder) which was partially used to help build the location data for this pack
